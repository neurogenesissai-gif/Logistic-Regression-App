# Logistic Regression Learning Web App

## Overview

This project is an interactive web-based system to learn and visualize Logistic Regression.

## Features

* Upload CSV dataset
* Data preprocessing
* Exploratory Data Analysis (EDA)
* Logistic Regression model training
* Prediction and evaluation

## Technologies Used

* Python
* Streamlit
* scikit-learn
* pandas, numpy
* matplotlib, seaborn

## How to Run

```bash
pip install -r requirements.txt
streamlit run app.py
```
